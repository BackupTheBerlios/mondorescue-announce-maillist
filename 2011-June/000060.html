Project mondorescue version 2.2.9.7 is now available

The project team is happy to announce the availability of a newest version of
mondorescue 2.2.9.7. Enjoy it as usual!

Maintenance version which fixes an annoying segmentation fault when including
or in some cases excluding directories. this version should also improve
multi-device support and computation of size of partitions. Extensive
description after the package list.

Now available at ftp://ftp.mondorescue.org
