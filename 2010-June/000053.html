Mindi 2.0.7.5 is now available

The project team is happy to announce the availability of a newest version of
mindi 2.0.7.5. Enjoy it as usual!

Now available at ftp://ftp.mondorescue.org

This version is a bug fix version for the issues reported around Xen kernel
detection which was failing in some contexts.
