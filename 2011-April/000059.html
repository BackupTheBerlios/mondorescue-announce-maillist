Project mondorescue version 2.2.9.6 is now available

The project team is happy to announce the availability of a newest version of
mondorescue 2.2.9.6. Enjoy it as usual!

this version should fix the core dumped encountered by users with the curses
interface, improve Arch Linux support, solve again the problem around size of
boot media for Debian 6, Fedora 14 and the likes, and deb dependencies around
wodim, genisimage.

Now available at ftp://ftp.mondorescue.org
