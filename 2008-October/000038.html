Project MondoRescue version 2.2.7 is now available

The project team is happy to announce the availability of a newest version of
MondoRescue 2.2.7. Enjoy it as usual!

Now available at ftp://ftp.mondorescue.org

Among the news coming with this version, you need to note that /dev/shm usage
is gone, syslinux static files are gone, new modules are supported in mindi for
Lenny and OpenSuSE 11.0, -E is taking devices also and a compatibility issue
format problem has been fixed with pre-2.2.5 versions.
